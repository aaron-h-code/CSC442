(b) How to build:
- Unzip the zip file.
- Enter into the "Code" folder through terminal.
- type "bash ./build.sh"
- If you want to rebuild the project, please delete the "bin" folder.

(c) How to run:
- Enter into the "Code" folder through terminal.
- To run examples: type "bash ./run.sh"
- Type task index according to the bash hints.
- For each task, we provide three test cases. For some cases, we provide several paramters for the user to specify. Please refer to the bash hints.

(d) If you use ssh to a cycle machine, please add the flag "-X".
